API开放对接文档：

邮件接口：

  • 请求方式：get
  • 请求参数：
    1. title=主题
    2. content=内容
    3. from=发件人邮箱(不可修改)
    4. to=收件人邮箱
  • 请求实例：http://39.105.204.201/api/sendmail/?title=主题&content=内容&from=yfhadmin@163.com&to=2113778814@qq.com

注册接口(有严重风险)：

  • 请求方式：get
  • 请求参数：
    1. username=用户名
    2. password=密码
    3. vip=是否vip
    4. score=初始积分
  • 请求实例：http://39.105.204.201/api/register/?username=user1&password=123456&vip=False&score=100

登录接口(有严重风险)：

  • 请求方式：get
  • 请求参数：
    1. username = 用户名
    2. password = 密码
  • 请求实例：http://39.105.204.201/api/login/?username=user1&password=123456
